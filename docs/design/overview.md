# 제품 개요

## 한 줄 정의
로컬 마크다운 파일을 빠르게 읽고 편집하는 **경량 데스크톱 리더 & 에디터**. Windows 우선, Microsoft Store 배포, 100% 오프라인.

## 목표
- **경량**: 빠른 실행, 낮은 메모리, 작은 번들.
- **리더 + 에디터**: 실시간 미리보기 분할 뷰.
- **워크스페이스**: 폴더 가져오기 + 가상 폴더로 가져온 파일만 계층 구성.
- **풍부한 마크다운**: 표·체크리스트·각주·수식(KaTeX)·다이어그램(mermaid)·코드 하이라이트 등 최대한 다양한 기능.
- **파일 친화**: 이미지 표시, 파일/폴더 열기, 최근, 즐겨찾기, 파일 감시(외부 변경 자동 반영).
- **내보내기**: HTML(자기완결), PDF.
- **UX**: 다크/라이트 + 종이질감 등 확장형 테마, 한국어/영어.

## 범위 밖 (현재)
- 실시간 협업·클라우드 동기화(오프라인 우선 원칙).
- 모바일(향후 Tauri 모바일 검토).

## 비기능 요건
- 대용량 md에서도 입력 지연 최소화(에디터 뷰포트 렌더 + 렌더 워커 + 디바운스).
- Windows 11 기본 지원, Windows 10 검토.
- 네트워크 없이 전 기능 동작(모든 자산 번들, 원격 호출 차단).

## 수익화 방향
- 초기 무료, 추후 **유료화 전환** 가능(기능 게이팅 또는 유료 등재/인앱결제).
- 배포되는 앱은 **상업적 용도 사용 허용** (자세한 내용은 [EULA](../legal/EULA.md)).
