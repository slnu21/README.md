# 개인정보 처리방침 — 초안

> ⚠️ 초안. 배포 전 검토.

**md-reader**는 **100% 오프라인**으로 동작한다.

- **수집 안 함**: 앱은 사용자 문서·파일 내용·이용 기록을 외부 서버로 전송하지 않는다.
- **로컬 저장만**: 워크스페이스 구조·최근·즐겨찾기·설정은 사용자 기기의 로컬 저장소(SQLite)에만 저장된다.
- **네트워크**: 기본적으로 외부 네트워크 호출을 하지 않는다(CSP로 원격 요청 차단). 향후 선택적 온라인 기능 도입 시 본 방침을 갱신하고 사용자에게 고지한다.
- **제3자 분석·광고 SDK 없음.**
