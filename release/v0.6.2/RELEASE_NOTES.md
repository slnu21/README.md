# README.md v0.6.2

가볍고 100% 오프라인인 마크다운 리더 & 에디터. Windows x64.

v0.6.1 실사용 후속 보완 — 드래그 피드백·탭 우클릭 메뉴 추가와 스크롤 동기화·파일 연결 버그 수정.

## 개선 · 추가

### 드래그 피드백
- 워크스페이스 트리와 열린문서 탭을 드래그하면 커서에 **항목 칩(아이콘+이름)**이 따라와 무엇을 옮기는지 한눈에 보입니다.
- 워크스페이스 **드래그 = 항상 '이동'**으로 통일. 가져온 폴더의 하위 개별 파일/폴더는 '폴더째 이동하는 단위'라 개별 이동이 안 되며, 잡으면 **금지 표시**로 안내합니다. (참조 추가가 필요하면 개별 파일 우클릭 → "워크스페이스에 참조 추가".)
- 가져온 폴더의 **폴더 행에 마우스를 올리면** 묶음 전체가 함께 강조됩니다.

### 탭 우클릭 메뉴
- 열린문서 탭에서 우클릭 → **워크스페이스에 추가 · 탭 닫기 · 다른 탭 모두 닫기 · 모든 탭 닫기 · 파일 위치 열기 · 경로 복사**. 저장 안 된 탭이 있으면 일괄 닫기 전에 저장/버림을 확인합니다.

## 수정

### 스크롤 동기화 안정화
- 에디터에 포커스를 두고 스크롤할 때 미리보기가 **문서 맨 위로 반복해서 튀던** 진동 문제 수정.

### 파일 연결 실행
- 탐색기에서 `.md`를 더블클릭했을 때 파일이 **조용히 탭에만 추가되고 활성화되지 않던** 문제 수정 — 이제 항상 그 파일이 바로 보입니다(콜드/웜 스타트 모두).
