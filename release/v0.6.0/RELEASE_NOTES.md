# README.md v0.6.0

가볍고 100% 오프라인인 마크다운 리더 & 에디터. Windows x64.

키보드 내비게이션 · 리더 UX · 전역 찾기·바꾸기 · 작성 도구 중심의 대규모 업데이트.

## 새로운 기능

### 작성 (T1)
- **서식 단축키**(토글) — Ctrl+B 굵게 · Ctrl+I 기울임 · Ctrl+E 인라인 코드 · Ctrl+K 링크
- **자동 목록 이어쓰기** — `-`/번호/체크박스, 빈 항목에서 Enter 시 목록 종료
- **괄호·백틱 자동 닫기**(따옴표 제외) · **스마트 붙여넣기**(선택 위 URL → 링크)
- **상태바** — 커서 줄:열 · 선택 글자수 · 읽기 시간

### 내비게이션 (T2)
- **명령 팔레트**(Ctrl+Shift+P) — 모든 명령을 퍼지 검색·실행
- **파일 퀵오픈**(Ctrl+P) — 워크스페이스 파일명 퍼지 검색
- **파일 타입 구분** — 마크다운은 전용 아이콘·강조, 비문서(이미지·바이너리)는 흐리게

### 리더 UX (T4)
- **이미지 라이트박스** · **리딩(집중) 모드** · **프레젠테이션 모드**(전체화면 슬라이드, `---` 분할)
- **양방향 스크롤 동기화**(편집 ↔ 미리보기) · **리딩 폭**(좁게/보통/넓게)

### 찾기·바꾸기 (T3)
- **워크스페이스 전역 찾기·바꾸기** — 리터럴/정규식, 파일별 미리보기·선택(기본=현재 파일), 파괴적 확인, 미저장 편집 보호

### 이식성 (T5) · 에디터 메뉴 (T6)
- **HTML 클립보드 복사** · **워크스페이스 내보내기/가져오기(JSON)**
- **에디터 커스텀 우클릭 메뉴** — 잘라내기/복사/붙여넣기 + 서식

### 데이터 안전 · 수정
- **닫기 전 저장 확인** · **세션 복원**(열린 탭) · **자동저장**(옵션)
- 상대경로 이미지 표시 개선(data URI 인라인) · 설정 톱니바퀴 아이콘 · 편집 선택 강조 · 머메이드 flowchart 라벨 · 워크스페이스 상호작용 버그 수정
