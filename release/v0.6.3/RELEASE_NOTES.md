# README.md v0.6.3

가볍고 100% 오프라인인 마크다운 리더 & 에디터. Windows x64.

v0.6.2 실사용 후속 수정 — 파일을 열자마자 '미저장'으로 표시되던 문제를 해결한 패치 릴리스.

## 수정

### 열자마자 '미저장'으로 표시되던 문제
- 파일을 **열기만 해도** 탭에 미저장 점과 "Unsaved" 상태가 뜨던 문제를 수정했습니다. 이제 **실제로 편집했을 때만** 미저장으로 표시됩니다.
- 파일이 외부(다른 편집기 등)에서 바뀌어 앱이 새로 불러온 뒤에도 해당 탭이 미저장으로 잘못 표시되던 문제도 함께 해결했습니다.
- 미저장 표시가 정확해지면서, 닫을 때 뜨던 불필요한 "저장하시겠습니까?" 확인도 사라집니다.

> 편집 중인 내용이 미저장으로 잡히지 않는 일은 없습니다 — 사용자 입력은 그대로 감지됩니다.
