# README.md v0.5.0

경량 마크다운 **리더 & 에디터** · Windows · Microsoft Store · 100% 오프라인.

v0.1 이후 누적된 v0.2–v0.5 기능을 통합한 릴리스입니다.

## 새 기능

**리치 미리보기 (v0.2)**
- 코드 하이라이트(highlight.js), 수식(KaTeX → MathML), mermaid 다이어그램, 각주·체크박스·콜아웃·표 등 markdown-it 플러그인 세트, 아웃라인/TOC, 문서 내 찾기/바꾸기.

**워크스페이스 & 검색 (v0.2·v0.4)**
- 가상 폴더로 흩어진 문서를 원하는 구조로 정리(생성·추가·이름변경·드래그 이동/재정렬).
- 즐겨찾기 최상단 고정, [워크스페이스 | 최근] 사이드바 탭.
- SQLite FTS5 기반 전역 검색, 즐겨찾기·최근·설정 영속.

**편집 경험 (v0.3)**
- 파일 드래그앤드롭 열기, 편집/미리보기 분할 폭 조정, 에디터/미리보기 확대·축소, 편집 위치에 맞춘 미리보기 스크롤 동기화.
- 글꼴 변경 — 시스템 프리셋 + 번들 고급 폰트(Lora · JetBrains Mono · 나눔명조 · Pretendard, 모두 OFL·오프라인).
- 아웃라인 우측 오버레이(호버·핀·투명도), 3테마(light/dark/paper).

**내보내기 (v0.5)**
- **HTML** — 이미지·폰트까지 파일에 내장한 자기완결 `.html`(다른 PC·브라우저에서도 동일하게 열림).
- **PDF** — OS 인쇄 대화상자(‘Microsoft Print to PDF’)로 저장.

**파일 연결 (v0.5)**
- `.md`·`.markdown`을 README.md로 열기(‘연결 프로그램’ 등록 + 더블클릭). 실행 중이면 기존 창에 새 탭으로 열림.

**기타 (v0.5)**
- 기본 우클릭(브라우저) 메뉴 억제로 데스크톱 앱다운 사용감. 에디터 텍스트 메뉴·워크스페이스 우클릭 메뉴는 유지.
